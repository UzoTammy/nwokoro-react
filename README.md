# Nwokoro React — Personal Portfolio Site

A modern personal portfolio website built with React, TypeScript, and Chakra UI.

## Tech Stack
- **Framework:** React with TypeScript
- **UI Library:** Chakra UI
- **Build Tool:** Vite
- **Language:** TypeScript

## Features
- Responsive design with Chakra UI components
- Modern single-page application architecture
- Fast development with Vite hot module replacement
- Type-safe codebase with TypeScript

## Getting Started

```bash
# Clone the repository
git clone https://github.com/UzoTammy/nwokoro-react.git
cd nwokoro-react

# Install dependencies
npm install

# Run development server
npm run dev
```
